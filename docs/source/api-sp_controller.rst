API sp_controller
==================

Requirements
-------------

* Turn off SmartPlugs
* Turn on SmartPlugs
* Queries filtered by: ID, row, column

------

* iotoad.org/api/in/sp_controller/{ID}
* iotoad.org/api/in/sp_controller/row/{row}
* iotoad.org/api/in/sp_controller/column/{column}
